# GCWARE Portfolio

Modern portfolio website built with Next.js 15, React 19, and Tailwind CSS 4.

## Tech Stack

- **Framework:** Next.js 15
- **UI Library:** React 19
- **Styling:** Tailwind CSS 4
- **Animations:** Anime.js
- **Icons:** Lucide React
- **Deployment:** Railway
- **Domain:** gcwarecr.com
